# 📊 Customer Churn Prediction using Machine Learning

## 🧠 Project Overview

Customer churn refers to customers who stop using a company's service. This project aims to build a machine learning model that predicts whether a customer is likely to churn based on their usage patterns, subscription details, and demographic information.

This project follows a complete data science pipeline and is designed to simulate a real-world business problem.

---

## 🎯 Objectives

* Understand the end-to-end machine learning workflow
* Analyze customer behavior and identify churn patterns
* Build predictive models to classify churn
* Compare multiple models to find the best performer
* Generate insights that can help businesses reduce churn

---

## 📌 Problem Statement

Given customer data such as tenure, monthly charges, contract type, and services used, predict whether a customer will leave the service (churn) or continue.

---

## 📊 Dataset Information

* Dataset: Telco Customer Churn Dataset
* Source: IBM Sample Dataset (Kaggle)
* Records: ~7000 customers
* Features: Demographics, account information, services subscribed
* Target Variable:

  * **Churn = Yes (1)**
  * **Churn = No (0)**
